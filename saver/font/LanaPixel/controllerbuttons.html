If you view this page with the LanaPixel font installed, you will be able to see the various controller button glyphs, which are mapped in the Private Use Area of the font. You can use these symbols by copy+pasting them. If you need the Unicode value, you can see it by hovering over the symbol.
Feel free to use these to display controller inputs in your game! These probably aren't as good as full-colour button sprites, but perhaps they'll suit your needs.
